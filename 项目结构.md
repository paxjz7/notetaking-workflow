# 项目结构

```
ai-research-workflow/
├── 📁 核心程序文件
│   ├── config.py           # 配置管理和提示词模板 (3.4KB)
│   ├── llm_client.py       # LLM客户端和工作流逻辑 (6.8KB)
│   ├── search_client.py    # 搜索客户端 (Web + PubMed) (13KB)
│   └── main.py             # 主程序和CLI界面 (10KB)
│
├── 📁 配置和依赖
│   ├── requirements.txt    # Python依赖包清单 (209B)
│   ├── .env.example       # 环境变量配置示例 (322B)
│   └── setup.py           # Python包安装脚本 (1.8KB)
│
├── 📁 测试和工具
│   ├── test_workflow.py   # 完整的测试脚本 (6.2KB)
│   └── quick_start.sh     # 一键启动脚本 (4.7KB, 可执行)
│
├── 📁 文档
│   ├── README.md          # 项目主要说明文档 (4.8KB)
│   ├── INSTALL.md         # 详细安装指南 (1.8KB)
│   ├── PROJECT_SUMMARY.md # 项目总结和技术细节 (5.1KB)
│   ├── workflow_analysis.md # 原n8n工作流分析 (4.9KB)
│   ├── 项目完成清单.md    # 功能清单和交付说明
│   └── 项目结构.md        # 本文件
│
├── 📁 运行时生成目录
│   └── results/           # 程序运行时创建，存放输出结果
│       ├── research_*.json  # 研究数据文件
│       └── report_*.md     # 研究报告文件
│
└── 📁 虚拟环境 (快速启动时创建)
    └── venv/              # Python虚拟环境
```

## 🚀 使用文件说明

### 必须配置的文件
1. **`.env`** - 从`.env.example`复制并配置API密钥
2. **`requirements.txt`** - 安装Python依赖

### 程序入口
- **`main.py`** - 主程序入口
- **`quick_start.sh`** - 一键启动脚本

### 测试文件
- **`test_workflow.py`** - 功能测试脚本

### 帮助文档
- **`README.md`** - 完整使用说明
- **`INSTALL.md`** - 安装指南
- **`项目完成清单.md`** - 功能清单

## 📝 模块说明

### config.py
- 环境变量管理
- LLM提示词模板
- 配置验证函数
- 常量定义

### llm_client.py
- LLM客户端封装
- 4阶段工作流实现
- OpenAI和本地LLM支持
- 异步处理逻辑

### search_client.py
- 网络搜索客户端
- PubMed学术搜索
- 搜索结果解析
- 批量搜索管理

### main.py
- CLI界面实现
- 程序主逻辑
- 结果保存和格式化
- 错误处理

这个项目结构清晰，模块化设计，便于维护和扩展。每个文件都有明确的职责，遵循了良好的软件工程实践。
